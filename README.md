# 利用方法

・第三者が投稿した内容にコメントを残すことができ、ハッピーの連鎖に繋げることができる

毎日嫌なことや不満に思うことをつぶやくより、楽しいことや嬉しいことをつぶやく癖をつけると、
気持ちが前向きになるということに気づいたので、前向きなことしか呟けないSNSを開発しようと思いました。
